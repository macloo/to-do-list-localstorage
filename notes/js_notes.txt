Splice:
newDeck.splice(num, 1);
Removes the item with index number "num" from the array newDeck, i.e. newDeck[num] is removed. The "1" limits the removal to a single item.
After this, the indexes of course change down from right to left, so I must go from the right-hand side, i.e. larger indexes down to smaller ones.

indexes.pop();
Removes last item (rightmost item) from the array indexes

THIS WORKS:
while (indexes.length > 0) {
    var i = indexes.pop();
    myArray.splice(i, 1);
}



// listen for button click to remove checked To Do items
$('.removeAll').click(function(e) {
    // get rid of the two buttons to avoid second click
    $('.removeAll').remove();
    // get index numbers from all checked items
    var indexes = handleCheckboxes();
    // remove those items from myArray
    while (indexes.length > 0) {
        var i = indexes.pop();
        myArray.splice(i, 1);
    }
    // write myArray out to localStorage
    localStorage.setItem( storageKey, JSON.stringify(myArray) );
    $('#toDoDataRow').hide();
    writeOutToDoList();
});
